## BalanceOS Brand Rules

BalanceOS is a premium nutrition management product.

Brand feeling:

- serious
- clean
- calm
- precise
- high-control
- modern
- professional
- not childish
- not noisy
- not motivational-first

The product should feel like:

- a nutrition control system
- a smart daily dashboard
- a professional tracking tool
- a calm health operating system

It should not feel like:

- a generic diet app
- a weight-loss challenge app
- a childish wellness game
- an over-motivational coach
- a cluttered calorie counter

## Visual Identity

Core visual language:

- white / off-white background
- green accent
- soft shadows
- rounded cards
- clean typography
- large readable numbers
- calm spacing
- minimal but premium UI

Green should communicate:

- health
- progress
- balance
- freshness
- control

Avoid:

- too many colors
- neon overload
- childish illustrations
- emojis as icons
- aggressive red/green guilt messaging
- cluttered dashboard cards

## Voice and Tone

User-facing tone should be:

- clear
- direct
- mature
- helpful
- calm
- non-judgmental

Avoid phrases like:

- "You failed"
- "You ruined the day"
- "Start over tomorrow"
- "Don’t give up!"

Prefer factual balance language:

- "נשאר להיום"
- "נאכל עד עכשיו"
- "יעד יומי"
- "איזון שבועי"
- "חלבון שנותר"
- "התקדמות"

Expert users should get fewer explanations and more data.
Guided users can get explanations, but the tone must stay mature.

## Home Dashboard Brand Rules

The Home Dashboard is the brand’s main product moment.

It should communicate immediately:

- control
- clarity
- daily progress
- calm confidence

The dashboard must be:

- numbers-first
- visually clean
- easy to scan in 3 seconds
- not overloaded with text
- not motivational-first

Main hierarchy:

1. Calories remaining
2. Eaten / target
3. Protein and macros
4. Add meal action
5. Meals / shortcuts
6. Settings access

## My Pantry Brand Rules

My Pantry should feel like a personal nutrition database.

It should communicate:

- organization
- precision
- speed
- control

Products should feel structured and reliable:

- product name
- brand
- values per 100g
- barcode if available
- clean product cards

Do not make pantry feel like a shopping app.
It is a personal tracking database.

## Settings Brand Rules

Settings should feel like a calm control center.

It should not be one long compressed form.

Use:

- grouped cards
- clear rows
- focused sub-screens
- clean hierarchy
- enough spacing
- professional tone

Danger actions must be visually separated and clear.
